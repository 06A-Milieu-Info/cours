# Module Excel

Dans ce module, les **notes de cours sont regroupées dans un seul classeur Excel**.  
Chaque **feuille du classeur correspond à une section du cours**, accompagnée d’un **exercice pratique**.

> ⚠️ **Important — Notes et exercices Excel**  
> Les notes de cours et les exercices sont fournis dans un **classeur Excel**.  
> 
> Avant de commencer, **télécharge une copie** et ouvre-la dans **l’application de bureau Excel**.  
> 
> **Étapes :**  
> 1. **Fichier**  
> 2. **Créer une copie**  
> 3. **Télécharger une copie**  
> 
> ⚠️ Le travail dans le navigateur peut limiter certaines fonctionnalités d’Excel.
